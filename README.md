# Installing stuff:
1. Install brew `https://brew.sh/`
2. Install zsh `brew install zsh`
3. Install tmux `brew install tmux`
4. Set zsh as your shelll `sudo echo "$(which zsh)" >> /etc/shells` & `chsh -s $(which zsh)`
5. Use the config files from this repo
